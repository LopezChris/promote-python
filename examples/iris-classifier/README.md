## Iris Flower Classification

Project structure:

```
├── README.md
├── deploy.py
├── helpers
│   └── getclass.py
├── objects
│   └── model_weights.pkl
├── requirements.txt
└── train.py
```

### Instructions

In a terminal shell run:

```bash
$ pip install promote

# first train the model (this will generate the model_weights.pkl file)
$ python train.py

# lastly, deploy the model
$ python deploy.py
```
