## Hello World
### Overview

This model accepts a `{"name": "your_name"}` and returns "Hello your_name!"

**Project structure:**

```
├── README.md
├── main.py
└── requirements.txt
```

### Instructions

In a terminal shell run:

```bash
$ pip install requirements.txt

# lastly, deploy the model
$ python main.py
```

### Example input:

```
{"name": "colin"}
```

### Result

```
{'response': 'Hello colin!'}
```
